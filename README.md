# 🌽 Chef Precision: Gestão de Custos & Receitas Inteligentes

Um Webbook avançado focado na sazonalidade de festas juninas, que funciona como um **Minisistema de Gestão de Produção (ERP Lite)** para empreendedores culinários.

## 🚀 Funcionalidades de Inteligência

### 💰 Calculadora de Custos em Tempo Real
*   **Ficha Técnica Automatizada:** O sistema calcula o custo exato de cada receita baseado nos preços de mercado inseridos pela cliente.
*   **Margem de Lucro:** Permite a visualização rápida do custo por porção, facilitando a precificação correta dos produtos.

### ⚖️ Escalonador Dinâmico (Scalability)
*   **Ajuste de Porções:** Algoritmo que recalcula automaticamente todas as dosagens de ingredientes caso a cliente queira duplicar, triplicar ou fracionar a produção.
*   **Precisão Operacional:** Elimina o erro humano no cálculo manual de grandes encomendas.

### 📝 Automação de Compras
*   **Gerador de Lista de Insumos:** Com base nas receitas selecionadas para a produção do dia, o sistema consolida todos os itens e gera uma lista de compras organizada por categoria.

## 🛠️ Stack Tecnológica
*   **Frontend:** React (Interface reativa para cálculos instantâneos).
*   **Lógica de Dados:** JavaScript (Algoritmos de conversão de medidas e cálculo de CMV).
*   **Persistência:** Supabase.
*   **Mobile First:** Design totalmente otimizado para uso em cozinhas e supermercados.

## 🧠 Valor de Processo
Este projeto demonstra minha habilidade em lidar com **lógica matemática complexa**, **relacionamento entre tabelas de insumos e produtos** e a criação de ferramentas que trazem previsibilidade financeira para o usuário.
